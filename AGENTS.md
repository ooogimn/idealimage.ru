# AGENTS RULES

## Терминальные команды (Shell)

- Агенту разрешено выполнять терминальные команды в этой среде.
- Любые терминальные команды выполняются только после явного разрешения пользователя.
- По умолчанию агент сначала запрашивает подтверждение, затем выполняет команды.
- Без разрешения пользователя терминальные команды не выполняются.
